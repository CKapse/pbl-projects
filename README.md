# DAA Assignments - MIT World Peace University

A collection of Design and Analysis of Algorithms assignments 
implemented in C++.

## University
MIT World Peace University, Pune
B.Tech Computer Science Engineering

## Algorithms Implemented

### Divide and Conquer
- Merge Sort
- Quick Sort
- Binary Search

### Greedy Algorithms
- Huffman Coding
- Job Sequencing with Deadlines
- Fractional Knapsack
- Prim's Algorithm (MST)
- Kruskal's Algorithm (MST)
- Dijkstra's Shortest Path

### Dynamic Programming
- 0/1 Knapsack
- Longest Common Subsequence (LCS)
- Matrix Chain Multiplication
- Floyd Warshall

### Backtracking
- N-Queens Problem
- Graph Coloring
- Subset Sum

### Graph Algorithms
- Breadth First Search (BFS)
- Depth First Search (DFS)
- Topological Sort
- Cycle Detection

### Sorting and Searching
- Bubble Sort
- Selection Sort
- Insertion Sort
- Heap Sort

### Hashing
- Hash Table Implementation
- Collision Handling
